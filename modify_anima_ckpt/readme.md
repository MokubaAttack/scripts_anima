# modify_anima_ckpt
It is a script that the ckpt that doesn't be loaded in diffusers_anima makes to be loaded.
If you modified diffusers_anima by my suggestion, this is not necessary.
## requirements
python modules
```
pip install git+https://github.com/hdae/diffusers-anima.git
pip install FreeSimpleGUI pyperclip
```
## How to use
1. Run this script.
2. Select the checkpoint files ( .safetensors file ).
3. Input the output path ( .safetensors file ).
4. Click run button.
5. After a while, the output file is generated.
